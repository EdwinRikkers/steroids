Hello world!

This file is located in www/index.html. It's just like a regular PhoneGap app.

Now it's time to put your app on Steroids.

Edit the file config/application.coffee to enable tabs by uncommenting few preconfigured lines: steroids.config.tabBar.enabled, steroids.config.tabBar.tabs and the array that defines the tabs.

Then, hit enter in the console where $ steroids connect is running, and this will get more interesting.
