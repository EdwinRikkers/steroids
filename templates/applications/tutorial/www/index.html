Hello world!

This file is located in www/index.html. It loads only a basic CSS file and both the Cordova and Steroids.js libraries. (Note that cordova.js is served from outside the project files to ensure that the version compatible with the current version of Steroids is used.)

If you have an existing PhoneGap project, you can simply place it in the www/ folder and enjoy the Steroids toolset, e.g. $ steroids simulator and $ steroids deploy. (Write $ steroids in your terminal to see the full list of available commands.)

Let's not stop there, though. Let's put your project on Steroids.

Edit the config/application.coffee file to enable tabs by uncommenting few preconfigured lines: steroids.config.tabBar.enabled, steroids.config.tabBar.tabs and the array that defines the tabs.

Then, hit enter in the console where $ steroids connect is running, and this will get more interesting.
