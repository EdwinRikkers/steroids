This page is generated from app/views/cures/index.html, this time with a different layout, app/views/layouts/cures.html. If a layout file with the same name as the current resource exists, it will be used instead of the generic layouts/application.html layout.

Steroids supports automatic compilation of CoffeeScript and SASS files: our controller is at app/controllers/cures.coffee and our resource-specific stylesheet is at www/stylesheets/cures.scss.

Steroids has an internal make command, which creates complete .html documents by merging the layouts and associated views, compiles CoffeeScript and SASS files and copies everything in the www/ and app/ to the dist/ folder. The make command is run when you press enter in the terminal running $ steroids connect. You can see the end result in the dist/ folder.
