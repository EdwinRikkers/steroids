Awesome, welcome!

Now, this page is a Steroids style resource from

app/views/pills/index.html

and it uses a layout from

app/views/layouts/application.html.

We've also enabled the tab bar: the Welcome tab shows our previous www/index.html document, and the Internet tab shows Google's front page.

If you look at the code for the app/views/pills/index.html document, you'll note that the controller for our index is initiated by calling PillsController.index();. This allows us to have a shared controller .js file for all views of a certain resource.

In addition, this page is served from http://localhost (instead of via the file:// protocol), which is part of our secret sauce.

One instant benefit of this: if you look at the app/views/layouts/application.html file, you'll notice that we don't have to use explicit relative paths for our .css and .js files, e.g. we can say /stylesheets/applcation.css instead of ../../stylesheets/application.css.

We've also included the Ratchet CSS library to give nice formatting to our document, and the Hammer.js library for handling touch events.
